# ESILV AI Track Handbook (Mock)

This is a **mock handbook** used for an offline-friendly GraphRAG MVP demo.

## 2024 (Core)

### AI101 — Introduction to Artificial Intelligence

- Year: 2024
- Projects: build a tiny rule-based assistant; discuss ethical considerations; implement a simple search-based planner.
- Prerequisites: none

### ML201 — Machine Learning Foundations

- Year: 2024
- Projects: train linear models; evaluate with cross-validation; write a short report on bias/variance.
- Prerequisites: AI101

### DL301 — Deep Learning

- Year: 2024
- Projects: implement an MLP; train a small CNN; compare optimizers; write an experiment logbook.
- Prerequisites: ML201

## 2025 (Advanced)

### NLP310 — Natural Language Processing

- Year: 2025
- Projects: text classification; embeddings; sequence labeling; mini RAG demo.
- Prerequisites: DL301

### MLOps320 — ML Systems & Deployment

- Year: 2025
- Projects: packaging; model registry; monitoring; reproducible pipelines.
- Prerequisites: ML201

### CAP400 — Capstone Project

- Year: 2025
- Projects: a full end-to-end AI product prototype with a written report and a demo.
- Prerequisites: DL301 and MLOps320

## Explicit prerequisite chains

- AI101 -> ML201 -> DL301
- AI101 -> ML201 -> DL301 -> NLP310
- AI101 -> ML201 -> MLOps320 -> CAP400
